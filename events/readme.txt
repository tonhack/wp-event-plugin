=== Event Listing Plugin ===

Event Listing helps listing Events.


== Description ==
Event Listing Plugin provide custom category and post type. it has title, description, event date, thumbanail and multiple categories.


== Installation ==

Upload events folder into plugin and then go to wordpress back panel and activate it and enjoy.

Add [event_listing] shortcode to the page you want to see and your are good to go.
